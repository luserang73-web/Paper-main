# Sweat-Induced Dermatosis (Miliaria-like Sweat Disease)

**Author:** Adam Hong (Hong, Adam)  
**Affiliation:** Independent Researcher  
**Location:** Las Vegas, NV, USA  

---

## Overview

This repository contains the primary academic manuscript and related materials for the study of **Sweat-Induced Dermatosis**, a sweat-related dermatological condition traditionally referred to in Korean as *땀병*.

The work documents clinical observations, symptom progression, environmental and physiological triggers, and proposes a refined conceptual framework for understanding sweat-induced skin inflammation distinct from conventional miliaria classifications.
